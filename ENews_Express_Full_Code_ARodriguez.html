1. Do the users spend more time on the new landing page than the existing landing page?¶

Perform Visual Analysis¶

Step 1: Define the null and alternate hypotheses¶

H0: The average time spent on the new landing page is equal to the average time spent on the existing landing page

HA: The average time spent on the new landing page is not equal to the average time spent on the existing page.

Step 2: Select Appropriate test¶

Two-tailed t-test

Step 3: Decide the significance level¶

Significance level 0.05

Step 4: Collect and prepare data¶

In [17]:
from scipy.stats import ttest_ind # Calling independent samples t-test function from scipy.stats

# Filtering dataset into two groups based on separate variables
new = df[df['landing_page'] == 'new']['time_spent_on_the_page']
old = df[df['landing_page'] == 'old']['time_spent_on_the_page']

Step 5: Calculate the p-value¶

In [18]:
# Running the t-test, calculating p-value
t_stat, p_value = ttest_ind(new, old, equal_var=False)
print("t-statistic:", t_stat)
print("p-value:", p_value)

t-statistic: 3.7867702694199856
p-value: 0.0002784762450333098

Step 6: Compare the p-value with $\alpha$¶

0.0002 < 0.05

Step 7: Draw inference¶

Since the p-value is less than 0.05, we reject the null hypothesis that the average time spent on the existing landing page is equal to the average time spent on the new landing page. Because the t-statistic is greater than 0, we can say that the average time spent on the new page is greater than the average time spent on the existing page with statistical significance of 0.05.

A similar approach can be followed to answer the other questions.

Performing Multiple Comparison test To identify which group differs from the others the null hypothesis is:

$𝐻_0: 𝜇_1=𝜇_2 \text{ and } 𝜇_1=𝜇_3 \text{ and } 𝜇_2=𝜇_3$

and the alternative hypothesis is:

$𝐻_a: 𝜇_1\neq𝜇_2 \text{ or } 𝜇_1\neq𝜇_3 \text{ or } 𝜇_2\neq𝜇_3$

Conclusion and Business Recommendations¶

The E-News Express sought to understand if the design of their new landing page was effective in gathering new subscribers.

We ran a two-tailed t-test to find out if the average time spent on the new page was significantly different from average the time spent on the existing page. We found the average time spent on the new page to be greater than the existing page with a statistical significance of 0.05.

We conducted a two-proportion z-test and found out if the conversion rate for the new page was significantly higher than that of the existing page with a statistical significance of 0.05.

We used the chi-square test of independence to determine if conversion status depended on preferred language and found that it does not with a statistical significance of 0.05.

Finally, we conducted ANOVA between the mean time spent on the new page according to preferred language and determined that while the means differ, the difference is not satistically significant.

Statistical evidence strongly suggests that the new landing page is effective in gathering new subscribers to E-News Express. While engagement levels seem to differ by language, it is not enough to suggest a change in their current approach to different language speakers.

If at all, I might suggest conducting a survey to see if there is another language that could be included in order to increase current engagement levels and customer base.

Examples
--------

    The simplest way to use nbconvert is

            > jupyter nbconvert mynotebook.ipynb --to html

            Options include ['asciidoc', 'custom', 'html', 'latex', 'markdown', 'notebook', 'pdf', 'python', 'qtpdf', 'qtpng', 'rst', 'script', 'slides', 'webpdf'].

            > jupyter nbconvert --to latex mynotebook.ipynb

            Both HTML and LaTeX support multiple output templates. LaTeX includes
            'base', 'article' and 'report'.  HTML includes 'basic', 'lab' and
            'classic'. You can specify the flavor of the format used.

            > jupyter nbconvert --to html --template lab mynotebook.ipynb

            You can also pipe the output to stdout, rather than a file

            > jupyter nbconvert mynotebook.ipynb --stdout

            PDF is generated via latex

            > jupyter nbconvert mynotebook.ipynb --to pdf

            You can get (and serve) a Reveal.js-powered slideshow

            > jupyter nbconvert myslides.ipynb --to slides --post serve

            Multiple notebooks can be given at the command line in a couple of
            different ways:

            > jupyter nbconvert notebook*.ipynb
            > jupyter nbconvert notebook1.ipynb notebook2.ipynb

            or you can specify the notebooks list in a config file, containing::

                c.NbConvertApp.notebooks = ["my_notebook.ipynb"]

            > jupyter nbconvert --config mycfg.py

To see all available configurables, use `--help-all`.
